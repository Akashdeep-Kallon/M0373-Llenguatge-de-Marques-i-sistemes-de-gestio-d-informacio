1. Sobre el display INLINE-BLOCK
    1. Cóm podem situar una caixa al costat de l’altre?

      2. Assignant display: inline-block a les caixes. Això permet que es comportin com elements en línia però conservin les propietats de bloc (amplada, alçada).

    2. Què passa si una caixa no hi cap al costat?

      3. Passa a la línia següent automàticament, com si fos text que no cap en una línia.

    3. Perquè serveix la propietat “vertical-align”?

      4. Per alinear verticalment els elements inline o inline-block respecte a la línia base, al mig, a dalt, etc. Exemple: vertical-align: middle.

    4. Com podem centrar una caixa dins d’una altre?

      5. Si és inline-block, pots usar text-align: center al contenidor. Si és bloc, millor usar Flexbox o Grid per centratge més precís.

  2. Sobre el display FLEX-BOX
    1. Cóm podem crear files?

      2. Amb display: flex i flex-direction: row (per defecte). Si vols columnes, usa flex-direction: column.

    2. Perquè serveix flex-wrap?

      3. Permet que els elements es distribueixin en múltiples línies si no hi caben. Exemple: flex-wrap: wrap.

    3. Perquè serveixen les propietats flex-grow, flex-shrink i flex-basis?
      • flex-grow: defineix com creix un element si hi ha espai extra.
      • flex-shrink: defineix com es redueix si no hi ha prou espai.
      • flex-basis: defineix la mida inicial abans de distribuir l’espai.
