LLENGUATGE DE MARQUES, HTML

Grupo 1

· Información de H1

Los elementos de cabecera implementan seis niveles de cabecera del documento es el más importante, y el menos importante. Un elemento de cabecera describe brevemente el tema de la sección que presenta. La información de cabecera puede ser utilizada por los agentes usuarios, por ejemplo, para construir una tabla de contenidos para un documento automáticamente.

· Información de HEADER

El elemento HTML Header (<header>) representa un grupo de ayudas introductorias o de navegación. Puede contener algunos elementos de cabecera, así como un logotipo, un formulario de búsqueda, un nombre de autor y otros componentes.

· Información de UL

El elemento <ul> en HTML significa "unordered list" o lista no ordenada. Se utiliza para agrupar elementos en forma de lista, pero sin un orden numérico. En lugar de números, los ítems suelen aparecer con viñetas (●, ■, ○), aunque esto puede cambiar con estilos CSS.

· Información de IMG

La etiqueta <img> se utiliza para insertar imágenes en una página web. Es una etiqueta auto-cerrada, lo que significa que no necesita una etiqueta de cierre como otras (</img> no existe).

· Información de alt (atributo)

El atributo alt (abreviación de "alternative") se utiliza dentro de la etiqueta <img> para proporcionar un texto alternativo que describe la imagen.

Grupo 2

· Información de NAV

La etiqueta <nav> se utiliza para definir una sección de navegación dentro de una página web. Es una etiqueta semántica introducida en HTML5 que indica que el contenido dentro de ella contiene enlaces de navegación.

· Información de OL

La etiqueta <ol> crea una lista ordenada, donde los elementos se numeran automáticamente. Es útil para mostrar pasos o clasificaciones.

· Información de LI

La etiqueta <li> define cada elemento dentro de una lista, ya sea ordenada (<ol>) o no ordenada (<ul>).

· Información de A

La etiqueta <a> se utiliza para crear enlaces (hipervínculos) que permiten navegar entre páginas o abrir recursos externos.

· Información de href (atributo)

El atributo href se usa dentro de la etiqueta <a> para indicar la dirección (URL) del destino del enlace.

Grupo 3

· Información de ARTICLE

La etiqueta <article> representa contenido independiente y autocontenido, como una noticia, entrada de blog o comentario. Puede incluir títulos, párrafos, imágenes, etc.

· Información de SECTION

La etiqueta <section> se usa para agrupar contenido relacionado dentro de una página. Cada sección puede tener su propio encabezado.

· Información de P

La etiqueta <p> define un párrafo de texto. Es una de las etiquetas más comunes en HTML.

· Información de BR

La etiqueta <br> se usa para insertar un salto de línea dentro de un texto, sin crear un nuevo párrafo.

· Información de TITLE

La etiqueta <title> define el título de la página web que aparece en la pestaña del navegador.

· Información del atributo title

El atributo title proporciona información adicional que aparece como un tooltip cuando se pasa el cursor sobre un elemento.

Grupo 4

· Información de FOOTER

La etiqueta <footer> representa el pie de página de una sección o documento. Suele contener información de contacto, derechos de autor, enlaces secundarios o notas legales.

· Información de ASIDE

La etiqueta <aside> se usa para contenido complementario o secundario, como barras laterales, anuncios, enlaces relacionados o notas.

· Información de TABLE

La etiqueta <table> se utiliza para crear tablas que organizan datos en filas y columnas. Es útil para mostrar información estructurada.

· Información de ROW

La etiqueta <tr> (table row) define una fila dentro de una tabla HTML. Cada fila puede contener una o más celdas (<td> o <th>).

· Información del atributo colspan

El atributo colspan permite que una celda ocupe varias columnas en una tabla. Es útil para combinar celdas horizontalmente.

· Información del atributo rowspan

El atributo rowspan permite que una celda ocupe varias filas en una tabla. Se usa para combinar celdas verticalmente.
